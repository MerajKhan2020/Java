Accessing the Manage Assets page

  • Click on the Asset button located on the left hand side panel of the program and ensure you are in the manage asset page by clicking on the manage assets tab


2.1 Add Asset


To add a new asset to the program, perform the following steps


Steps:

  • Ensure you are in the Manage Asset page - instructions to access this is located on top of this page
  • Enter the details of the new asset you would like to add ensuring that all required fields marked with an asterix(*).


  • Click on the Add button when finished entering details


  • If successful, a pop up box will appear alerting the user that the asset has been added to the program succesfully


  • If a pop up box didnt appear or something did not work as expected, scroll down to the bottom of this page to see the troubleshooter

2.2 Search Asset


To search for an existing asset, perform the following steps


Steps:

  • Ensure you are in the Manage Asset page - instructions to access this is located on top of this page
  • Enter a Asset Name in the Asset Name text box


  • Click on the Search Button


  • The table is then loaded with your search results


  • Simply click on the row of the asset to bring up its stored information and to undertake further actions on that record



2.3 Update Asset


To update an existing asset, perform the following steps


Steps:

  • Ensure you are in the Manage Asset page - instructions to access this is located on top of this page
  • Follow 2.2 Search Asset instuctions above to search for an existing asset to update
  • When the record is searched, clicked on and all its information is brought up, simply proceed to change any values that are needed


  • Once the updated information is complete, simply click on the Update button


  • If the record was succesfully updated, a pop up box will appear stating that the record was updated succesfully


  • If a pop up box didnt appear or something did not work as expected, scroll down to the bottom of this page to see the troubleshooter



2.4 Delete Asset


To delete an existing asset from the program, perform the following steps


Steps:

  • Ensure you are in the Manage Asset page - instructions to access this is located on top of this page
  • Follow 2.2 Search Asset instuctions above to search for an existing asset to delete
  • When the record is searched, clicked on and all its information is brought up

  • Simply click on the Delete button to delete the record from the system


  • If the record was succesfully deleted, a pop up box will appear stating that the record was deleted succesfully


  • If a pop up box didnt appear or something did not work as expected, scroll down to the bottom of this page to see the troubleshooter



Troubleshooting/Notes:

This section will help you to troubleshoot any common problems encountered during the programs operation. It is also used to give you any hand tips that may help you when using the program

  • When you see the below pop up box appear, it means that there is a text box that contains a alphabetical letter when there is meant to be a number. Please ensure when you are filling out the fields on the page, that you are inputting the correct formats. Textfields that require a number are: Number of Items.


  • When you see the below pop up box appear, it means that there is a required field that you missing. Please ensure when you are filling out all required fields. Textfields that are required are: Name, Type, Number of Items and Location


  • When you see the below pop up box appear, it means that the name typed into the Asset Name text box doesnt exist in the system. Ensure the name is spelt correctly or try searching for another name that exists.


  • When you see the below pop up box appear, it means that you have not searched for a record to update or delete. Ensure you type in a valid Name in the Name field, click on search and select a record from the table



  • **Handy Hint: If you leave the asset name text box blank and click search, the table will show all Assets currently stored in the system.

    • As you can see, leave the asset name box blank and click on the search button



    brings up the whole list of assets stored in the program



  • **Handy Hint: You are able to sort the results from the table by clicking on the column heading
