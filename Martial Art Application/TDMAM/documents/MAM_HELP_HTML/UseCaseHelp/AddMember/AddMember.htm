Accessing the Manage Member page

  • Click on the Members button located on the left hand side panel of the program and ensure you are in the manage member page by clicking on the manage member tab.


3.1 Add Member


To add a new member to the program, please perform the following steps.


Steps:

  • Ensure you are in the Manage Member page - instructions to access this is located on top of this page.
  • Enter the details of the new member you would like to add ensuring that all required fields marked with an asterix(*) are filled in before proceeding.


  • Click on the Add button when finished entering details.


  • If successful, a pop up box will appear alerting the user that the member has been added to the program succesfully.


  • If a pop up box didnt appear or something did not work as expected, scroll down to the bottom of this page to see the troubleshooter.

3.2 Search Member


To search for an existing member, please perform the following steps.


Steps:

  • Ensure you are in the Manage Member page - instructions to access this is located on top of this page.
  • Enter a Members Last Name in the Last Name text box.


  • Click on the Search Button located next to the Last Name text box.


  • The table is then loaded with your search results.


  • Simply click on the row of the member to bring up its stored information and to undertake further actions needed on that members record.

 

  • If a pop up box didnt appear or something did not work as expected, scroll down to the bottom of this page to see the troubleshooter.



3.3 Update Member


To update an existing asset, perform the following steps.


Steps:

  • Ensure you are in the Manage Member page - instructions to access this is located on top of this page.
  • Follow 3.2 Search Member instuctions above to search for an existing member to update before proceeding.
  • When the record is searched, clicked on and all its information is brought up, simply proceed to change any values that are needed in the relevant page elements (ie text boxes, drop downs etc).


  • Once the updated information is entered, simply click on the Update button.


  • If the record was succesfully updated, a pop up box will appear stating that the record was updated succesfully.


  • If a pop up box didnt appear or something did not work as expected, scroll down to the bottom of this page to see the troubleshooter.




3.4 Delete Member


To delete an existing member from the program, perform the following steps.


Steps:

  • Ensure you are in the Manage Member page - instructions to access this is located on top of this page
  • Follow 3.2 Search Member instuctions above to search for an existing member to delete before proceeding
  • When the record is searched and clicked on, all of its information is brought up in the relevant page elements (ie text boxes, drop downs etc).

  • Simply click on the Delete button to delete the record from the system.


  • If the record was succesfully deleted, a pop up box will appear stating that the record was deleted succesfully.


  • If a pop up box didnt appear or something did not work as expected, scroll down to the bottom of this page to see the troubleshooter.
