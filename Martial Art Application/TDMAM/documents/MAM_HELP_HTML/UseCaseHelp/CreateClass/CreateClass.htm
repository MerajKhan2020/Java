Accessing the Manage Class page

  • Click on the Class button located on the left hand side panel of the program and ensure you are in the manage Class page by clicking on the manage Class tab


4.1 Create Class


To add a new Class to the program, perform the following steps


Steps:

  • Ensure you are in the Manage Class page - instructions to access this is located on top of this page
  • Enter the details of the new Class you would like to add ensuring that all required fields marked with an asterix(*).


  • To add a member list for the Class, click on the Get button next to the Memberlist


  • The memberlist pop up will appear on the screen


  • Fill in the details for the member list and click on the shown table to select members to add to the Class


  • Click on the save button


  • the memberlist id is returned to manage Class page


  • Click on the Add button when finished entering details


  • If successful, a pop up box will appear alerting the user that the Class has been added to the program succesfully


  • If a pop up box didnt appear or something did not work as expected, scroll down to the bottom of this page to see the troubleshooter

4.2 Search Class


To search for an existing Class, perform the following steps


Steps:

  • Ensure you are in the Manage Class page - instructions to access this is located on top of this page
  • Enter a Class Name in the Class Name text box


  • Click on the Search Button


  • The table is then loaded with your search results


  • simply click on the row of the asset to bring up its stored information and to undertake further actions on that record



4.3 Update Class


To update an existing Class, perform the following steps


Steps:

  • Ensure you are in the Manage Class page - instructions to access this is located on top of this page
  • Follow 4.2 Search Class instuctions above to search for an existing Class to update
  • When the record is searched, clicked on and all its information is brought up, simply proceed to change any values that are needed


  • Once the updated information is complete, simply click on the Update button


  • If the record was succesfully updated, a pop up box will appear stating that the record was updated succesfully


  • If a pop up box didnt appear or something did not work as expected, scroll down to the bottom of this page to see the troubleshooter



4.4 Delete Class


To delete an existing class from the program, perform the following steps


Steps:

  • Ensure you are in the Manage Class page - instructions to access this is located on top of this page
  • Follow 4.2 Search Class instuctions above to search for an existing Class to delete
  • When the record is searched, clicked on and all its information is brought up

  • simply click on the Delete button to delete the record from the system


  • If the record was succesfully deleted, a pop up box will appear stating that the record was deleted succesfully


  • If a pop up box didnt appear or something did not work as expected, scroll down to the bottom of this page to see the troubleshooter



Troubleshooting/Notes:

This section will help you to troubleshoot any common problems encountered during the programs operation. It is also used to give you any hand tips that may help you when using the program

  • When you see the below pop up box appear, it means that there is a required field that you missing. Please ensure when you are filling out all required fields. Textfields that are required are: Name, start time, finish time, location, class type, memberlist


  • When you see the below pop up box appear, it means that the name typed into the Class Name text box doesnt exist in the system. Ensure the name is spelt correctly or try searching for another name that exists.


  • When you see the below pop up box appear, it means that you have not searched for a record to update or delete. Ensure you type in a valid class name in the class name field, click on search and select a record from the table



  • **Handy Hint: If you leave the class name text box blank and click search, the table will show all classes currently stored in the system.

    • As you can see, leave the class name box blank and click on the search button



    Brings up the whole list of members stored in the program



  • **Handy Hint: You are able to sort the results from the table by clicking on the column heading
