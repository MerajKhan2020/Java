Accessing the Manage Group page

  • Click on the Group button located on the left hand side panel of the program and ensure you are in the manage group page by clicking on the manage group tab


6.1 Create Group


To add a new group to the program, perform the following steps


Steps:

  • Ensure you are in the Manage Group page - instructions to access this is located on top of this page
  • Enter the details of the new Group you would like to add ensuring that all required fields marked with an asterix(*).


  • To add a member list for the group, click on the Get button next to the Memberlist


  • The memberlist pop up will appear on the screen


  • Fill in the details for the member list and click on the shown table to select members to add to the group


  • Click on the save button


  • the memberlist id is returned to manage group page


  • Click on the Add button when finished entering details


  • If successful, a pop up box will appear alerting the user that the Group has been added to the program succesfully


  • If a pop up box didnt appear or something did not work as expected, scroll down to the bottom of this page to see the troubleshooter

6.2 Search Group


To search for an existing group, perform the following steps


Steps:

  • Ensure you are in the Manage Group page - instructions to access this is located on top of this page
  • Enter a Group Name in the Group Name text box


  • Click on the Search Button


  • The table is then loaded with your search results


  • Simply click on the row of the Group to bring up its stored information and to undertake further actions on that record



6.3 Update Group


To update an existing group, perform the following steps


Steps:

  • Ensure you are in the Manage Group page - instructions to access this is located on top of this page
  • Follow 2.2 Search Group instuctions above to search for an existing Group to update
  • When the record is searched, clicked on and all its information is brought up, simply proceed to change any values that are needed


  • Once the updated information is complete, simply click on the Update button


  • If the record was succesfully updated, a pop up box will appear stating that the record was updated succesfully


  • If a pop up box didnt appear or something did not work as expected, scroll down to the bottom of this page to see the troubleshooter



6.4 Delete Group


To delete an existing group from the program, perform the following steps


Steps:

  • Ensure you are in the Manage Group page - instructions to access this is located on top of this page
  • Follow 2.2 Search Group instuctions above to search for an existing Group to delete
  • When the record is searched, clicked on and all its information is brought up

  • simply click on the Delete button to delete the record from the system


  • If the record was succesfully deleted, a pop up box will appear stating that the record was deleted succesfully


  • If a pop up box didnt appear or something did not work as expected, scroll down to the bottom of this page to see the troubleshooter
