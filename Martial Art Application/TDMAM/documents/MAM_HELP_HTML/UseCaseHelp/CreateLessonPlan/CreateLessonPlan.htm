7.1 Create Lesson Plan


The steps below are to assist the user of the Martial Arts Manager application on how to CREATE/ADD a new Lesson Plan record in the database.


Steps:

  • Once the user starts the application and navigates to the Home page, they then must click on the Classes Tab.
  • Following this, the user is then to click the Manage Lesson Plan button in order to open the Lesson Plan page.
  • The user then must enter the minimal information for the lesson plan to be created, which is Lesson Plan Name.
  • If the user desires to input more details that is fine to do so otherwise details can be updated at a later stage.
  • Once the user inputs the desired details for the new lesson plan, the user then presses the "Add New Plan" button to add the new lesson plan record in the database.

    7.0 Lesson Plan - Getting to the Landing Page


    The below steps are to guide the user to the landing page of Lesson Plan.

    Steps:

    • When the user starts the program, the first page should be as displayed. They must then press the Start button and can either choose a database at this stage or at a later stage as shown.

    • This will land the user on the home page.

    • They then must click on the Classes Tab and then the Manage Lesson Plan button in order to open the Lesson Plan page.

    7.3 Update Lesson Plan


    The steps below are to assist the user of the Martial Arts Manager application on how to UPDATE a record from the Lesson Plan table in the database.


    Steps:

      <<<<<<< HEAD
    • Once the user starts the application and navigates to the Home page, they then must click on the Classes Tab.
    • Following this, the user is then to click the Manage Lesson Plan button in order to open the Lesson Plan page.
    • The user must than type a Lesson Plan Name in its field.
