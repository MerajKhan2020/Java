5.1 Create New Database

To create a new Database, please follow the steps below.

Pre-Condition:
- N/A

Steps:

  • Click on "Utilities" button from the left side of the window and then click on "Create New Database" tab.

  • In the name field, type a name for the database and click on "Create Database"

  • Chose a location where you would like to save the new Database file

  • Once the location is confirmed, a confirmation message will apprear showing the database name and path.


5.2 Check Table list of newly created/existing Database

You can check the list of tables of a newly created database or existing one.

Steps:

  • Click on "show table list" button and select a DB file.

  • The table view willpopulate the list of tables the database consists of.



Notes:

  • When chosing a name of the Database, please ensure it unique.
  • When chosing a path for the newly created DB file, please ensure that the path does not contain a DB file which has the same name. If itdoes, the file will get overwritten with the new database.
