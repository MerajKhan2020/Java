9.1 Create Rank Structure


The steps below are to assist the user of the Martial Arts Manager application on how to CREATE/ADD a new Rank record in the database.


Steps:

  • Once the user starts the application and navigates to the Home page, they then must click on the Ranks Tab.
  • Following this, the user is then to click the Manage Ranks button in order to open the Ranks page.
  • The user then must enter the minimal information for the rank to be created, which is Rank Name.
  • If the user desires to input more details that is fine to do so otherwise details can be updated at a later stage.
  • Once the user inputs the desired details for the new lesson, the user then presses the "Add" button to add the new rank record in the database.

    Steps:
