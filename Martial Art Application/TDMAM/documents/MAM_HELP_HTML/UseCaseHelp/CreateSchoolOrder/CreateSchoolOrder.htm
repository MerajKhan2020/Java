Accessing the Manage School Orders page

  • Click on the Orders button located on the left hand side panel of the program and ensure you are in the manage orders page by clicking on the manage orders tab


10.1 Create School Order


To add a new school order to the program, perform the following steps


Steps:

  • Ensure you are in the Manage School Order page - instructions to access this is located on top of this page
  • Enter the details of the new school order you would like to add ensuring that all required fields marked with an asterix(*).


  • Click on a member row in the tableview below to select members to add to the order.


  • Click on the Add button when finished entering details


  • If successful, a pop up box will appear alerting the user that the School Order has been added to the program succesfully


  • If a pop up box didnt appear or something did not work as expected, scroll down to the bottom of this page to see the troubleshooter

10.2 Search School Order


To search for an existing School Order, perform the following steps


Steps:

  • Ensure you are in the Manage School Order page - instructions to access this is located on top of this page
  • Click on the calender icon next to the school date and select a date for the new school order.


  • Click on the Search Button


  • The table is then loaded with your search results


  • Simply click on the row of the school order to bring up its stored information and to undertake further actions on that record

  • If a pop up box didnt appear or something did not work as expected, scroll down to the bottom of this page to see the troubleshooter



10.3 Update School Order


To update an existing School Order, perform the following steps


Steps:

  • Ensure you are in the Manage School Order page - instructions to access this is located on top of this page
  • Follow 2.2 Search School Order instuctions above to search for an existing school order to update
  • When the record is searched, clicked on and all its information is brought up, simply proceed to change any values that are needed

 

  • Once the updated information is complete, simply click on the Update button


  • If the record was succesfully updated, a pop up box will appear stating that the record was updated succesfully

 

  • If a pop up box didnt appear or something did not work as expected, scroll down to the bottom of this page to see the troubleshooter



10.4 Delete School Order


To delete an existing School Order from the program, perform the following steps


Steps:

  • Ensure you are in the Manage School Order page - instructions to access this is located on top of this page
  • Follow 2.2 Search School Order instuctions above to search for an existing School Order to delete
  • When the record is searched, clicked on and all its information is brought up

  • simply click on the Delete button to delete the record from the system


  • If the record was succesfully deleted, a pop up box will appear stating that the record was deleted succesfully

 

  • If a pop up box didnt appear or something did not work as expected, scroll down to the bottom of this page to see the troubleshooter



Troubleshooting/Notes:

This section will help you to troubleshoot any common problems encountered during the programs operation. It is also used to give you any hand tips that may help you when using the program

  • When you see the below pop up box appear, it means that there is a text box that contains a alphabetical letter when there is meant to be a number. Please ensure when you are filling out the fields on the page, that you are inputting the correct formats. Textfields that require a number are: SupplierId and all the Member Order fields.

 

  • When you see the below pop up box appear, it means that there is a required field that you missing. Please ensure when you are filling out all required fields. Textfields that are required are: Order Date, SupplierId and Member0

 

  • When you see the below pop up box appear, it means that the name typed into the Order Date text box doesnt exist in the system. Ensure the date was selected from the calender picker or try searching for another date that exists.

 

  • When you see the below pop up box appear, it means that you have not searched for a record to update or delete. Ensure you type in a valid Order Date in the Name field, click on search and select a record from the table
