11.1 Export Data Set


The steps below are to assist the user of the Martial Arts Manager application on how to EXPORT a database table into a CSV/Excel file.


Steps:

  • Once the user starts the application and navigates to the Home page, they then must click on the Utilities Tab.
  • This will then open on the Reports page automatically.
  • The user must then press the "Get Tables" button. This will load the tables in the dropdown menu where the user will need to select a table name.
  • Once a table name has been selected, the user the user must press the "Show Table" button.
  • Once the table is displayed on the screen, the user must press the "Export to File" button.
  • The application will now ask the user to select a file location and a file name to save the information to.
  • Once the user has chosen the location and file name, they must press okay to save the table information to a file properly.
  • Congratulations, you have successfully saved/exported a table information to a file.
