13.1 Mark Roll

To mark roll for a class, please follow the steps below.

Pre-Condition:
1. Class has been created.
2. Students has been added to the class.
3. Class name has to be unique. 

Steps:

  • Click on "Classes" button from the left side of the window and then click on "Mark Roll" tab.

 

  • Click on  
  • Select Database

  • Once the database is selected, Location and Classes drop down lists will be pre-poluated with data. To select a location, click on the drop down list and chose a location and click on it. Then click on Class drop down list and select a class to mark roll.

    •         

 

  • After selecting the Class, the application should populate the list of students in the class. Click on check boxes to mark student as present. Comments can be added related to the class in the Comment field.

  • Once the students are marked, click on Save to complete the mark roll action. An alert box will appear to confirm student rolls are marked.

  • To view marked student rolls, click on "View Marked Roll" Tab and type a class name in the search field and click on "Search".
  • A list of class names will appear in the table that matched the keyword. Note that a marked class will have the name appearing as "Class name - Marked".

13.2 Delete a Class:

Steps:

  • To delete a class, select the class in the table and click on "Delete".
  • An alert will appear to confirm the class has been deleted.
